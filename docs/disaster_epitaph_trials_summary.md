# 재앙의 임말소 — 3차 재판 핵심 모순·증거·논증 정리

> **이름 대응 (게임 기준):** 원고의 「김탈북」→ **이소은**, 1차 피고인 「바텐더」→ **소호**, 2·3차 피고인 → **앤서니(Anthony)**

---

## 1차 재판 — 클럽 VX 살인 (피고: 소호 / 증인: 이소은)

### 사건 개요

강남스타일 클럽에서 양진혁이 VX 중독으로 사망. 증인 이소은은 바텐더 소호를 누명하려 위증.

### 수집·제출 증거 (조사 단계)

| 증거 | 핵심 내용 |
|------|-----------|
| **양진혁 부검 기록** | 23:30경, VX 신경가스 중독 사망 |
| **이소은 진료 기록** | **오른손**에서 VX 검출 |
| **VX 정보** | 호흡 50mg / **피부 10mg** 치사량; VX2(합성 과정)는 부가 정보 |
| **VX 술잔 2개** | 각 잔 **20mg** VX (치사량 미만) |
| **CCTV (2장)** | 당시 클럽 상황 |

### 핵심 모순 ① — 「마신 양은 치사량이 아니다」

| 구분 | 내용 |
|------|------|
| **증인 주장** | 술 마시고 춤 → 불길한 기분 → 기절 → 깨어보니 병원, 양진혁 사망 |
| **모순** | 잔당 20mg은 **호흡·피부 기준 모두 치사량 미만** |
| **반박 증거** | VX 정보 + 술잔 검출량 |
| **논증** | 「한 번 마셔도 치사량이 아님」→ **음주만으로는 설명 불가** |

**후속:** 검사가 **양진혁 심장질환 + 의사 소견** 제출 → 「치사량은 아니어도 위험한 양 + 기저질환」으로 검찰 논리 보강.

### 핵심 모순 ② — 「오른손 VX = 양진혁이 쓰러지며 묻었다」

| 구분 | 내용 |
|------|------|
| **증인 주장** | 양진혁이 쓰러지며 독 술이 **오른손**에 묻음 |
| **질문** | 오른손 VX 검출 이유? |
| **반박 논점** | **이소은이 먼저 쓰러졌다**면, 양진혁이 손에 묻히기 **전**에 이미 오른손에 VX가 있어야 함 |

### 핵심 모순 ③ — 「손에 쏟은 뒤 춤 → 기절」 (결정타)

| 구분 | 내용 |
|------|------|
| **수정 진술** | 기억 상실 → 사실은 **춤 전** 취한 양진혁이 **오른손에 술을 전부 쏟음** → 손에 묻은 채 춤 → 독 기운 → **먼저** 쓰러짐 |
| **결정적 모순** | **피부 흡수 치사량 10mg**. 춤 전에 손에 쏟였다면 **춤을 추기 전 즉사**해야 함 |
| **반박 증거** | **VX 정보**(피부 10mg) + **진료 기록**(오른손 VX) + 수정 진술 자체 |
| **논증** | 「두꺼운 피부·장갑」 등 변명 → **붕괴 → 자백** |

**결과:** 이소은 자백 → 「바텐더는 죄 없다」→ 검사 재판 연기 신청.

### 1차 재판 흐름

1. 증인: 술 마시고 기절
2. 치사량? → 20mg/잔 → 치사량 아님
3. 심장질환 + 의사소견으로 검찰 보강
4. 오른손 VX = 양진혁이 묻힘
5. 먼저 쓰러진 건 누구?
6. 수정: 춤 전 손에 쏟음
7. 피부 10mg → 춤 전 즉사해야 함 → **모순**
8. **자백 · 재판 연기**

## 2차 재판 — 자율주행차 ‘해킹’ 살인 (피고: 앤서니)

### 사건 개요

1차 재판 후 이소은 연행 중 **자율주행차 사고로 사망**. YJ社 엔지니어 **앤서니**가 이소은 차량을 해킹해 살해한 혐의.

### 조사·제출 증거

| 구분 | 증거 | 핵심 내용 |
|------|------|-----------|
| **수집** | CCTV (지도) | 18:08:22, 급가속 → **교차로 급좌회전** → 가로등 충돌 |
| **수집** | 자율주행 기록 | **화재로 판독 불가** |
| **검사** | 앤서니 노트북 | **회사 서버 접속** 기록 |
| **검사** | 서버 로그 | 18:07:11 차량에 **「급가속 후 우회전」** 코드 송신 |
| **대화 중** | 앤서니↔임민수 카톡 | 양진혁 복수·바텐더(소호) 살해 의도 |

### 핵심 모순 ① — 「해킹 코드 = 우회전」 vs 「실제 = 좌회전」

| 구분 | 내용 |
|------|------|
| **검사 주장** | 서버 로그·노트북으로 앤서니 해킹 입증 |
| **변호 논증** | **CCTV: 실제 차량은 우회전** / **로그: 좌회전 명령** → **불일치** |
| **검사 재반박** | 임민수 소견서: Lidar 과민 → **예상 외 기동 시 센서 이상 → 반대 방향 회피** 가능 |
| **앤서니 반응** | 「임민수, 거짓말!」 |

### 핵심 모순 ② — 「동기 ≠ 유죄」

| 구분 | 내용 |
|------|------|
| **검사** | 카톡으로 **살해 동기** 확실 → 유죄 주장 |
| **변호·재판장** | **동기만으로는 유죄 불가** |
| **앤서니** | 해킹 실력은 자신뿐이나, **로그·노트북 조작**은 타인도 가능 |

**결과:** 증거·동기만으로 판결 곤란 → **재판 연기**.  
**엔딩:** 앤서니 「임민수 배신!」 / 임민수 등장 「해킹 없이도 가능한데」→ **3차 재판 예고**.

### 2차 재판 쟁점 요약

| 쟁점 | 검사 | 변호 |
|------|------|------|
| 해킹 여부 | 서버 로그 + 노트북 | CCTV **우회전** ≠ 로그 **좌회전** |
| 사고 메커니즘 | Lidar 오작동 소견 | **희박한 확률** |
| 유죄 근거 | 카톡 동기 | **동기 ≠ 범행** |
| 판결 | — | **연기** (증거 불충분) |

## 3차 재판 — 진범 규명 (피고: 앤서니 / 증인: 임민수)

### 사건 개요

연기 후 **CTO 임민수**를 증인으로 소환. 해킹이 아닌 **레이저 + 현장 부재**가 핵심.

### 추가·제출 증거

| 증거 | 제출 시점 | 핵심 내용 |
|------|-----------|-----------|
| **CCTV (심화 분석)** | 증인 진술 | 해킹 아님 → **적외선 레이저**로 Lidar 교란 |
| **앤서니 차량 위치 기록** | 증인 | 피해 차량 **미행** → 최적 위치에서 레이저 |
| **얼굴 식별 기록** | 증인 | 사고 직전 **앤서니 운전** 사진 |
| **양진혁 유언장** | 조력자 제안 | 앞장: 임민수에게 전부 / **뒷장: 비어 있음** (함정) |

### 핵심 모순 ① — 「앤서니가 운전했다」 vs 「그림자 방향」

| 구분 | 내용 |
|------|------|
| **임민수 주장** | 얼굴 식별 기록 = 사고 직전 **앤서니 운전** |
| **변호 논증** | 기록: 얼굴 **우측면 그림자** / CCTV: 사고 차·앤서니 차 **오른쪽에 해** → **조작** |
| **임민수 변명** | 좌우 반전 / 거울 / 구름 |
| **변호 재반박** | **운전석 위치**로 좌우 반전 아님을 입증 |

### 핵심 모순 ② — 「레이저·미행」 vs 조작 증거

| 구분 | 내용 |
|------|------|
| **임민수** | 레이저 + 미행 + 얼굴 기록 → **앤서니 범행** |
| **변호** | 얼굴 기록 **조작** → **전체 서사 붕괴** |
| **임민수** | 카톡(바텐더 살해 의도) 재제시 |
| **변호·재판장** | **증오·대화 ≠ 이번 범행**; **조작 증거로 유죄 만들려 한 이유** 추궁 |

**결과:** 임민수 당황 → 검사 **재연기** 신청 vs 변호 **지금 끝내야 함**.

### 핵심 모순 ③ — 「유언장 뒷장」함정 (자백 유도)

| 구분 | 내용 |
|------|------|
| **조력자 힌트** | **발상 전환** — 유언장 **뒷장** 확인 |
| **사실** | 뒷장 **공란** |
| **함정 질문** | 「뒷장에 *‘앞장 철회, Anthony에게 전 재산·지분’* 있다는 걸 아십니까?」 |
| **임민수 반응** | 멘붕 → **「사실 내가 죽였어」** 자백 |
| **앤서니** | 「유언장에 없던 말인데?」 |
| **변호** | **함정 자백이지만 성공** |

**판결:** 임민수 **체포** / 앤서니 **무죄**.

### 3차 재판 단계 요약

| 단계 | 쟁점 | 핵심 증거 | 결과 |
|------|------|-----------|------|
| 1 | 해킹 vs 레이저 | CCTV, 위치 기록 | 임민수: 레이저 주장 |
| 2 | 앤서니 운전 | 얼굴 식별 | **그림자·운전석** → 조작 |
| 3 | 살해 동기 | 카톡 | **동기 ≠ 범행** |
| 4 | 진범 | 유언장 함정 | **임민수 자백** |
| 5 | 판결 | — | **앤서니 무죄** |
